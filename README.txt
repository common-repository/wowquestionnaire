== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wow` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->WOW!Q screen to configure the plugin
